# Aydentify Demo Website

This project is a modern demo landing page built with Next.js and Tailwind CSS. It showcases a polished product-style website with sections for hero content, benefits, testimonials, pricing, FAQs, and a call-to-action footer.

## What this project includes

- A responsive landing page layout
- Custom section components for marketing content
- Styled buttons, cards, and reusable UI patterns
- A simple navigation experience for desktop and mobile

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open http://localhost:3000 in your browser.

## Project structure

- app/ - main app pages and global styles
- app/components/ - reusable UI components
- app/sections/ - landing page sections

## Notes

This is a demo website intended for presentation and visual design purposes, rather than a full production application.
